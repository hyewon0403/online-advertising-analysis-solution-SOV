## 온라인 광고 분석 솔루션 ‘SOV’ 런칭

### 프로젝트 소개
‘SOV’는 **머신러닝을 기반으로 광고 트렌드를 분석하여 제공하는 솔루션**입니다. 
광고주별 온라인 광고 정보를 모니터링할 수 있도록 실시간으로 수집된 광고 소재와 미디어렙사의 실 집행 데이터를 바탕으로 추정된 광고비 데이터를 제공합니다. 
또한, 솔루션 데이터를 기반으로 분석 리포트를 정기적으로 제작 및 배포했습니다.
